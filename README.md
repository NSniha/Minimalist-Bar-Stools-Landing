A minimalist, product-first e-commerce landing page concept for a bar stool brand — **StoolMart**. Built from scratch with HTML, CSS, and vanilla JavaScript, with a strong focus on typography, visual hierarchy, and a layout that puts the product front and center.

**Repo:** https://github.com/NSniha/Minimalist-Bar-Stools-Landing


---

## Design Approach

- **Typography-led hero** — an oversized "STOOLS" wordmark anchors the page and does the branding work instead of stock hero images.
- **Product-first layout** — one large hero product shot paired with a secondary product card, mimicking real e-commerce catalog pages rather than generic template sections.
- **Restrained color palette** — soft yellow accents against black/white keep focus on the product photography.
- **Category pills** (Wooden / Metal / Swivel) act as lightweight filters without needing a backend.
- **Conversion-focused CTA** — a single, high-contrast "Order Now" button rather than competing calls-to-action.

## Features

- Sticky top navigation with clear section links
- Hero section with bold typography and featured product image
- Secondary product highlight card
- Category quick-filter pills
- Cart and Sign In UI, structured for future e-commerce integration
- Fully responsive layout, pure CSS — no frameworks
- Zero dependencies — pure HTML/CSS/JS, loads instantly

## Getting Started

No build step, no dependencies. Clone and open directly:

```bash
git clone https://github.com/NSniha/Minimalist-Bar-Stools-Landing.git
cd Minimalist-Bar-Stools-Landing
```

Open `index.html` in your browser, or use a live server (e.g. VS Code Live Server) for auto-reload during editing.
